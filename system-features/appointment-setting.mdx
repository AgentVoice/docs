## Native Calendar

The Agent Calendar allows you to schedule and manage tasks for your AI agents. This helps organize your agent's activities and ensures timely follow-ups with contacts.

Access the calendar by navigating to the **Activity** section and clicking on the **Calendar** tab.

The calendar interface shows a monthly view with all scheduled tasks and a detailed task list for the selected day. If you have multiple agents, use the dropdown menu to view a specific agent's calendar.
